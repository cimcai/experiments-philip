# Experiments-philip


ants.html:  Ants are particles interacting with a lattice of values (pheremones), trying to move a T-shaped object through two barriers with openings, as demonstrated by real ants at https://www.youtube.com/watch?v=j9xnhmFA7Ao  The goal here is to try and find the simplest rules for the ants that allows them to solve this problem.

saccade.html:  Processes live webcam video on the GPU into a foveated view: a small region around the fixation point stays sharp while the periphery is progressively blurred. The fixation point jumps at random intervals (microsaccades and large saccades), mimicking eye movement. An optional cortical-magnification overlay shows a log-polar remap with a transient channel that fades when the scene is still. Gaze mode uses WebGazer.js to follow where you look on screen.
